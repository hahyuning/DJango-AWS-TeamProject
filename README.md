# Django-AWS
서울시 관련 통계/문화 정보를 제공하는 웹 사이트

## 개요
1. 참여 인원: 3명
<br>

2. 사용 기술
+ Python
+ Django
+ AWS - RDB, S3
+ Map API
+ JavaScript, JQuery, Ajax
<br>

3. 맡은 역할
+ MTV 패턴 기반의 모델 설계
+ AWS - RDB, S3 연동
+ 서울시 문화 정보 수집과 행사 정보 출력, 필터링
+ 지도 API를 연동
+ 게시판과 페이징 기능 구현
<br>
